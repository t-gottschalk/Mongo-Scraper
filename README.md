# Mongo-Scraper
UNC Bootcamp Homework

Created during UNC Coding Bootcamp. This assignment was to create an online web scraper app using Node, Express, Handlebars, Mongoose, and Cheerio while deploying on Heroku using mLab.

Technology used:

* Node.js
* Express
* NPM
* Mongoose
* Handlebars
* Body-parser
* Cheerio
* Request
* mLab
